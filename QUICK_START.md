## 🆘 Common Issues & Solutions

### Website Shows "Not Found" After Upload
**Solution:** Check that files are in `public_html/` folder, not in a subfolder

### Links Don't Work
**Solution:** Use relative paths (e.g., `about.html` not `/about.html`)

### Images Not Showing
**Solution:** Verify images are uploaded in `images/` folder and paths match

### Form Not Sending
**Solution:** Use FormSubmit.co or add PHP handler (see README.md)

### Mobile Menu Doesn't Expand
**Solution:** Clear browser cache and hard refresh (Ctrl+Shift+R)

### Styling Looks Wrong
**Solution:** Clear cache, verify CSS file uploaded, check file permissions
